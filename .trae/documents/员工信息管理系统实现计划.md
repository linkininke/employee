# 员工信息管理系统实现计划

## 技术栈

* **前端**：Vue 3 + Element Plus + Axios + Vite

* **后端**：Spring Boot 3 + Java 17

* **数据库**：MySQL 8 + MyBatis-Plus

* **认证**：Spring Security + JWT

* **API文档**：Swagger/OpenAPI

## 一、项目初始化

### 1. 后端项目初始化

* 使用 Spring Initializr 创建 Spring Boot 项目

* 添加依赖：Spring Web, Spring Security, MyBatis-Plus, MySQL Driver, JWT, Swagger

* 配置 application.yml：数据库连接、MyBatis-Plus、JWT 密钥等

### 2. 前端项目初始化

* 使用 Vite 创建 Vue 3 项目

* 安装依赖：Element Plus, Axios, Vue Router, Pinia（可选）

* 配置 vite.config.js

* 设置 Element Plus 按需导入

## 二、数据库设计

### 1. 员工表（employee）

* id: 主键，自增

* name: 姓名

* gender: 性别

* age: 年龄

* department: 部门

* position: 职位

* salary: 薪资

* hire\_date: 入职日期

* create\_time: 创建时间

* update\_time: 更新时间

### 2. 用户表（sys\_user）

* id: 主键，自增

* username: 用户名

* password: 密码（加密存储）

* create\_time: 创建时间

* update\_time: 更新时间

## 三、后端实现

### 1. 基础架构

* 实体类（Entity）：Employee, SysUser

* Mapper 接口：EmployeeMapper, SysUserMapper

* Service 层：EmployeeService, SysUserService

* Controller 层：EmployeeController, SysUserController, AuthController

### 2. 认证授权

* 实现 JWT 工具类：生成 token、解析 token

* 配置 Spring Security：

  * 自定义 UserDetailsService

  * 配置密码加密

  * 配置 JWT 过滤器

  * 配置权限规则

* 实现登录接口：/api/auth/login

* 实现获取当前用户信息接口：/api/auth/me

### 3. 员工管理 API

* 分页查询：GET /api/employees?page=1\&size=10

* 根据 ID 查询：GET /api/employees/{id}

* 添加员工：POST /api/employees

* 更新员工：PUT /api/employees/{id}

* 删除员工：DELETE /api/employees/{id}

* 批量删除：DELETE /api/employees/batch

### 4. 异常处理

* 自定义全局异常处理器

* 统一响应格式

### 5. API 文档

* 配置 Swagger/OpenAPI

* 为接口添加文档注释

## 四、前端实现

### 1. 基础架构

* 配置 Vue Router：

  * 登录页：/login

  * 首页：/dashboard

  * 员工列表：/employees

  * 员工详情：/employees/{id}

  * 添加员工：/employees/add

  * 编辑员工：/employees/edit/{id}

* 配置 Axios 拦截器：

  * 请求拦截：添加 token

  * 响应拦截：处理错误、刷新 token

* 封装 API 调用：

  * auth.js：登录、获取用户信息

  * employee.js：员工管理相关接口

### 2. 组件设计

* 登录组件：Login.vue

* 布局组件：Layout.vue（包含侧边栏、顶部导航、内容区域）

* 员工列表组件：EmployeeList.vue

* 员工表单组件：EmployeeForm.vue（用于添加和编辑员工）

* 员工详情组件：EmployeeDetail.vue

### 3. 页面实现

* 批量删除功能

#### 员工添加/编辑页

* 表单布局

* 提交按钮

* 取消按钮

#### 员工详情页

## 八、项目目录结构

### 后端目录结构

```
src/main/java/com/example/employee/
├── config/          # 配置类
├── controller/      # 控制器
├── entity/          # 实体类
├── mapper/          # Mapper 接口
├── service/         # 服务层
│   └── impl/        # 服务实现
├── utils/           # 工具类
└── EmployeeApplication.java  # 启动类
```

### 前端目录结构

```
src/
├── assets/          # 静态资源
├── components/      # 组件
├── router/          # 路由配置
├── services/        # API 封装
├── stores/          # 状态管理（Pinia）
├── views/           # 页面
├── App.vue          # 根组件
└── main.js          # 入口文件
```
